EXCEL

Excel é um editor de planilhas e é possível realizar cálculos matemáticos de forma automatizada, gerenciar a contabilidade de uma empresa, fazer relatórios, organizar dados em tabelas e até mesmo gerar recibos e notas fiscais.

A função principal do Excel é a de criar fórmulas e inserir cálculos, além de ser possível organizar os dados e configurar diversos tipos de gráfico.

O Excel por ter a fama de ser um utencilio dificil de se usar mas sua grande questão é a pratica e fuçação para o comprendelo.

ATALHOS DESCRIÇÃO
Shift+F8 Adicionar uma célula não adjacente ou um intervalo a uma seleção de células utilizando as teclas de direção.
Alt+Enter Iniciar uma nova linha na mesma célula.
Ctrl+Enter Preencher o intervalo de células selecionado com a entrada atual.
Alt+H Vá para a guia Página Inicial.
Alt+P Vá para a guia Layout da Página.
Alt+A Vá para a guia Dados.
Alt+M Vá para a guia Fórmula.
Ctrl+Alt+N Aplicar negrito.
Alt+N Vá para a guia Inserir.
